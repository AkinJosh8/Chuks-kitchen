# Chuks Kitchen

Chuks Kitchen is a food ordering web application that allows users to browse meals, add items to cart, and place orders seamlessly.

**Live Demo:**  
https://chukks-kitchen.netlify.app/

---

## Overview

Chuks Kitchen is a frontend-only food ordering system built using Vanilla JavaScript.  
It simulates a real-world food delivery flow including cart management, delivery details, payment processing, and order confirmation.

The application uses **LocalStorage** to manage cart data and order details.

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage (for state management)
- Netlify (Deployment)

---

## Features

### Authentication
- Sign Up Page
- Sign In Page

### Core Pages
- Home Page
- Explore Page (Browse meals)

### Cart System
- Add to Cart
- Increase / Decrease Quantity
- Remove Items
- Auto-update Cart Count
- Dynamic Cart Total Calculation

### Checkout Flow
- Delivery Details Page
- Payment Page
- Order Confirmation Page
- Auto-generated Order ID

### Storage
- Persistent cart using LocalStorage
- Delivery information saved locally
- Order ID stored per transaction
